# help

Show Astrape plugin usage guide.

## Activation
- Manual: `/astrape:help`

## Available Commands

| Command | Description |
|---------|-------------|
| `/astrape:ultrawork` | Maximum parallel execution |
| `/astrape:analyze` | Deep analysis mode |
| `/astrape:search` | Comprehensive search |
| `/astrape:plan` | Strategic planning |

## Keywords

Say these naturally in your prompt:
- `ultrawork` / `ulw` - Parallel execution
- `analyze` / `debug` - Deep analysis
- `search` / `find` - Comprehensive search
- `plan` - Strategic planning

## Agents

32 agents across 3 tiers (Haiku/Sonnet/Opus):
- architect, executor, explore, designer
- researcher, writer, qa-tester, scientist
- security-reviewer, build-fixer, code-reviewer

Use `-low`, `-medium`, `-high` suffixes for tier control.
